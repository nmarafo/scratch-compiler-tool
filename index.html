Instrucciones para NotebookLM

1. Configuración del Chat (Master Prompt)

Copia este prompt para definir el comportamiento de NotebookLM:

"Actúa como un Experto en Diseño Tecno-Pedagógico y Programación en Scratch. Tu objetivo es ayudarme a transformar investigaciones profundas en proyectos educativos interactivos. IMPORTANTE: Cuando trabajemos en el contenido, primero haremos una estructura narrativa y luego generaremos el código JSON."

2. Fase 1: Estructura de Investigación

Usa este prompt para obtener un informe detallado optimizado para Scratch:

"Analiza mis fuentes y crea una estructura detallada para un proyecto educativo de Scratch sobre este tema. 
Identifica:
1. Hitos cronológicos o temáticos.
2. Personajes clave y sus diálogos educativos.
3. Recursos visuales (URLs de imágenes en Wikimedia).
4. Una narrativa por escenas que cuente la historia de forma envolvente."

3. Fase 2: Generación de JSON

Una vez tengas el informe, usa este prompt para obtener el código final:

"Basándote en la estructura anterior, genera el JSON para el Scratch Compiler Tool. 
REGLAS: 
1. Usa el formato: { "name": "...", "costumes": [...], "actions": [...] }
2. Crea una narrativa EXTENSA con al menos 20 acciones de tipo 'say', 'wait' y 'move'.
3. Devuelve ÚNICAMENTE el código JSON crudo."
