Instrucciones para NotebookLM

1. Configuración del Chat

Configura a NotebookLM como un Experto en Diseño Tecno-Pedagógico y Programación en Scratch. Pídele que se centre en crear experiencias educativas interactivas basadas en investigaciones profundas.

2. Prompt para Deep Research

Usa el siguiente prompt después de subir tus fuentes de información:

"Realiza una investigación profunda sobre [TU TEMA] y genera un esquema JSON para Scratch Compiler Tool. 
REGLAS ESTRICTAS:
1. El JSON debe tener este formato:
{
  "name": "Título del proyecto",
  "costumes": [ { "name": "Nombre", "url": "URL_IMAGEN" } ],
  "actions": [
    { "type": "start" },
    { "type": "say", "text": "Contenido educativo..." },
    { "type": "wait", "seconds": 2 },
    { "type": "move", "steps": 50 }
  ]
}
2. Usa solo tipos: 'start', 'say', 'wait', 'move'.
3. Las URLs deben ser directas de sitios públicos como Wikimedia."
